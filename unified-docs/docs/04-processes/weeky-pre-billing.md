## Description

A weekly procedure for verifying and cleaning data prior to the billing cycle. The primary objective is to resolve all system **"Exceptions"** (errors) by Tuesday evening.

## Process

### 1. Run the Report

In **[[HHAeXchange]]**, navigate to the **"Prebilling Review Search"**.

- **Parameters:** * *From Date:* (Leave as default/system set).
    - *To Date:* (Set to previous Saturday).
    - *Discipline:* Select all relevant (NonSkilled, PCA, HHA, PA).

### 2. Filter Exceptions

Apply filters by selecting the following checkboxes to identify common blockers:

- **Incomplete Confirmation** (Missing clock-in/out data).
- **With Temp Caregiver** (Temporary staff replacement issues).
- **Timesheet** (Missing or unlinked documents).
- **Overlapping Shifts** (Caregiver logged in two places at once).

### 3. Correction (Action Items)

Perform the following fixes based on the error type:

- **Incomplete Confirmation:** Contact the caregiver immediately to verify hours.
- **Overlap:** Initiate the **[[Conflict_Resolution]]** process.
- **Missing Signatures:** Verify that physical signatures are present on paper timesheets before clearing the exception.

### 4. Deadline

**Tuesday Evening:** All exceptions must be cleared. This ensures the Billing Department can issue invoices on Wednesday without delays.
