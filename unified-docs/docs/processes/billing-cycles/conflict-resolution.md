# Conflict Resolution
## Scenario
The caregiver works for two agencies, and their hours overlap. This prevents payment.

## Solutions
1. **Request for an Explanatory Note**: The caregiver must write a statement explaining the discrepancy.
2. **Corrections**:
  - If the error is ours: Unbill/Rebill.
  - If the error is a competitor's: The caregiver must address it with them.
  - *Policy:* We try not to cancel billing if we have EVV confirmation.
3. **Contacts**: For all disputes, please contact Anastasiia Kes.
